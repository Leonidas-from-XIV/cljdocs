# cljdocs

A Clojure/ClojureScript amalgam to display the docs.

## Usage

```sh
# build JS code once
lein cljsbuild once
# or on every change
lein cljsbuild auto
# to get a REPL
lein cljsbuild once
# to start the server
lein ring server
```
